개요: 이 과제의 주요 목표는 Breast Cancer Wisconsin 데이터셋을 활용하여 유방암 양성과 음성을 분류하는 분류 분석 모델을 개발을 하는 것이다. 데이터를 훈련 및 테스트 셋으로 분할하고 로지스틱 회귀분석, 의사결정나무, 앙상블 기법, SVM 등을 활용하여 각 모델의 성능을 비교해볼 것이다.

wisconsin dataset출처 Breast Cancer Wisconsin 데이터셋은 종양의 특성에 관한 다양한 특징(세포 크기, 모양 등)과
양성(1) 또는 음성(0) 유방암 여부를 포함하고 있다.

id: 환자 아이디
diagnosis: 유방암 여부 (M: 악성, B: 양성)
radius_mean: 반지름(둘레의 중심에서 점까지의 거리 mean)
texture_mean: 유방 종양의 질감의 평균값
perimeter_mean: 유방 종양의 둘레의 평균값
area_mean: 유방 종양의 면적의 평균값
smoothness_mean: 유방 종양 표면의 평균값
compactness_mean: 컴팩트성(perimeter^2 / area - 1.0) 평균값
concavity_mean: 오목한 부분 (윤곽의 오목한 부분의 severity)
concave.points_mean: 오목한 점(윤곽의 오목한 부분의 수)
symmetry_mean: 유방 종양의 대칭성의 평균값
fractal_dimension_mean: 유방 종양의 프랙탈 차원의 평균값

데이터의 통계적인 정보와 분포는 해당 특성들의 평균, 중앙값, 최솟값, 최댓값 등을 통해 확인할 수 있다. 각 세포 특성에 관한 것이며 유방암의 악성(M)과 양성(B)을 구분하는 것이 주된 목적이다.

class(df$diagnosis)
## [1] "character"
df$diagnosis <- ifelse(df$diagnosis == "M", 0, 1)
df$diagnosis <- factor(df$diagnosis)  # 종속 변수 factor 형태로 변환

diagnosis 열이 “M”이면 0으로, 그렇지 않으면 1로 값을 변환한다. 여기서 “M”은 악성(Malignant)을 나타내며, 0으로 매핑되고, “B”는 양성(Benign)을 나타내며, 1로 매핑된다. diagnosis 열을 팩터(factor) 형태로 변환하였다.

# 데이터 분할 : train 70%, test 30% 
set.seed(123)
idx <- sample(1:nrow(df),nrow(df)*0.7,replace = FALSE)
train <-df[idx,]
test <- df [-idx,]

데이터셋은 train 70%, test 30%으로 분할하였다.
